# My Learning Roadmap

> A chronological record of my journey in computer science and software engineering.

This repository documents what I've learned each year, the technologies I've explored, and how my interests have evolved over time.

It is not a list of everything I know.
It is a timeline of how I learned.

---

## Philosophy

Learning is a long-term investment.

Instead of chasing every new technology, I focus on building strong fundamentals, practicing consistently, and improving a little every day.

This repository helps me:

- Track my progress
- Reflect on my learning journey
- Stay accountable
- Remember where I started
- Appreciate the people who helped me grow
